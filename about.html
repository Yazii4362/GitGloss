GitHub README를
더 쉽고 예쁘게

코딩 없이 Markdown 한 줄로 GitHub 프로필을 꾸밀 수 있는 무료 위젯 빌더예요. 60가지 이상의 템플릿 중에서 골라 바로 사용할 수 있어요.

✓ 완전 무료 ✓ 회원가입 불필요 ✓ 60+ 위젯 템플릿 ✓ Markdown 한 줄 삽입

지원하는 위젯 종류

📊
GitHub Stats Card
스타 수, 커밋 수, 레포지토리 수 등 핵심 통계를 카드 형태로 표시해요. 10가지 이상의 테마를 지원합니다.
🏷️
Tech Stack Badge
React, TypeScript, Node.js, Python, Go, Rust 등 수백 가지 기술 스택을 배지 형태로 나열해요.
👤
Profile Card
이름, 직함, 소개, 기술 스택을 하나의 카드에 담아 개발자 정체성을 한눈에 보여줘요.
🔥
Streak Counter
연속 커밋 일수와 총 기여 수를 시각적으로 표현해서 꾸준한 개발 활동을 어필할 수 있어요.
🔗
Links
GitHub, 블로그, LinkedIn, 이메일 등 SNS 링크를 버튼 형태로 모아서 보여줘요.
🎨
Banner
capsule-render, readme-typing-svg 등 외부 무료 API를 활용한 헤더 배너를 만들 수 있어요.

GitGloss를 사용해야 하는 이유

  • 🆓
    완전 무료
    모든 기본 위젯은 무료로 제공됩니다. 회원가입 없이 즉시 사용할 수 있어요.
  • ⚡
    설치 불필요
    별도 앱 설치나 서버 설정 없이 웹 브라우저에서 바로 사용합니다.
  • 🔄
    자동 업데이트
    github-readme-stats 등 실제 GitHub API와 연동되어 통계 데이터가 자동으로 최신 상태를 유지해요.
  • 🎨
    다양한 테마
    라이트 모드, 다크 모드, 뉴모피즘, 글래스모피즘 등 60가지 이상의 디자인 테마를 제공합니다.
  • 📝
    Markdown 호환
    GitHub README, GitLab, Notion, 개인 블로그 등 Markdown을 지원하는 모든 플랫폼에서 사용 가능해요.

자주 묻는 질문

빌더에서 위젯을 설정한 후 생성된 Markdown 코드를 복사하세요. GitHub에서 본인 아이디와 동일한 이름의 public 저장소를 만들고, 그 안의 README.md에 붙여넣으면 프로필 페이지에 바로 표시됩니다.
github-readme-stats 등 실제 GitHub API 기반 서비스를 사용하며, 캐시 정책에 따라 수 시간 단위로 자동 갱신됩니다. 별도 조작 없이 항상 최신 통계가 표시돼요.
기본 설정에서는 Public 저장소 데이터만 표시됩니다. Private 저장소 통계까지 포함하려면 github-readme-stats를 직접 Vercel에 배포하고 GitHub Personal Access Token을 설정해야 해요. GitGloss 빌더에서 생성한 코드의 URL을 본인 배포 주소로 교체하면 됩니다.
GitHub README는 이미지 태그로 위젯을 표시하기 때문에 모바일에서도 동일하게 보입니다. 다만 GitHub 모바일 앱에서는 일부 레이아웃이 다르게 보일 수 있어요.
Markdown 이미지 문법(![](url))을 지원하는 GitLab, Notion, 개인 블로그, 기술 문서 등 어디서든 사용할 수 있어요. HTML 탭의 <img> 태그를 사용하면 더 많은 플랫폼에서 활용 가능합니다.
